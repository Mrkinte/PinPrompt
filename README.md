## 🎲 为什么开发这款软件

- 作者是一名电气工程师，有时想在客户产线的工控机屏幕上显示一些注意事项，以前都是在图片上添加文字后替换背景，缺乏灵活性。最近刚好在学习C#，为了更好的学以致用，故而开发了此款软件，如有需要，欢迎下载体验。

## 🛠️ 钉提示 PinPrompt 的特点

- 支持显示多个提示信息，每个提示信息可单独设置参数。

- 提示信息可置顶显示，不被其他窗口覆盖。

- 支持开机自启动，开机后自动显示提示信息。

- 支持多种字体样式和颜色设置，能实现较为丰富的显示效果。
